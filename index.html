Project StreetXML

Progetto per la realizzazione di un programma che fornisce all'utente informazioni riguardanti meteo e posizione geografica di una località da lui scelta

Descrizione Progetto

Il programma lavora tramite la ricezione di file xml da dei web server esterni che forniscono informazioni al programma riguardo alla località scelta dell'utente e immessa in input.

Vengono utilizzati i seguenti web services:

  • Google maps API per le informazioni relative alla posizione geografica
  • OpenWeatherMap per le informazioni relative al meteo

Da ognuno dei web service viene ottenuto un file xml contenente le informazioni richieste, come latitudine e longitudine della località, continente e stato, temperature, condizione climatica come umidità, pressione atmosferica e molto altro. Per permettere la connessione ad internet in tutti i computer è stata implementata l'opzione per accedere tramite proxy, mediante autenticazione con password e nome utente

Realizzazione Progetto

Il programma è realizzato tramite il linguaggio di programmazione JAVA, sviluppato utilizzando la versione del JDK 1.7 tramite l'ambiente di sviluppo NetBeans. Per la parte riguardante l'XML viene utilizzata la libreria java XPath che permette di esplorare tramite uno pseudocodice i file XML che si ottengono dai vari web server Per quanto riguarda la grafica viene utilizzala la libreria grafica integrata in netbeans, Swing, basata su AWT.
